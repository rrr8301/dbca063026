    `IllegalStateException: Could not find a valid Docker environment`, which propagates as
    `ExceptionInInitializerError` in each test class that depends on the database, causing
    203 test failures. The original CI (GitHub Actions runner) has a Docker daemon available
    and Testcontainers can successfully launch the PostgreSQL container, so all tests pass.
