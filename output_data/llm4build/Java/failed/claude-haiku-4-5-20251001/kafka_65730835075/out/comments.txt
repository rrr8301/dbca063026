    directories or files and then expect an `IOException` to be thrown. Root ignores all
    file permission bits, so the I/O operations succeed, the expected exception is never
    thrown, and the assertions fail (`expected: <1> but was: <0>`, `expected: <2> but was: <0>`).
    The original CI's 8 failures are timing/flakiness-related tests and are not caused by
