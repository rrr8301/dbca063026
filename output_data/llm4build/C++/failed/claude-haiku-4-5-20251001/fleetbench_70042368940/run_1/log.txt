#7 36.86 Packages

Here is a list of packages that may be useful along with the default install:

#7 36.86 - `python#.#-gdbm`: provides the standard library `dbm.gnu` module
#7 36.86 - `python#.#-tk`: provides the standard library `tkinter` module
