#7 26.74 Packages

Here is a list of packages that may be useful along with the default install:

#7 26.74 - `python#.#-dev`: includes development headers for building C extensions
#7 26.74 - `python#.#-venv`: provides the standard library `venv` module
#7 26.74 - `python#.#-distutils`: provides the standard library `distutils` module
#7 26.74 - `python#.#-lib2to3`: provides the `2to3-#.#` utility as well as the standard library `lib2to3` module
#7 26.74 - `python#.#-gdbm`: provides the standard library `dbm.gnu` module
#7 26.74 - `python#.#-tk`: provides the standard library `tkinter` module
