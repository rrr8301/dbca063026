#7 57.39 Packages
#7 57.39 ========
#7 57.39 
#7 57.39 The packages provided here are loosely based on the debian upstream packages with some modifications to make them more usable as non-default pythons and on ubuntu.  As such, the packages follow debian's patterns and often do not include a full python distribution with just `apt install python#.#`.  Here is a list of packages that may be useful along with the default install:
#7 57.39 
#7 57.39 - `python#.#-dev`: includes development headers for building C extensions
#7 57.39 - `python#.#-venv`: provides the standard library `venv` module
#7 57.39 - `python#.#-distutils`: provides the standard library `distutils` module
#7 57.39 - `python#.#-lib2to3`: provides the `2to3-#.#` utility as well as the standard library `lib2to3` module
#7 57.39 - `python#.#-gdbm`: provides the standard library `dbm.gnu` module
#7 57.39 - `python#.#-tk`: provides the standard library `tkinter` module
#7 57.39
