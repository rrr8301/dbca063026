#7 34.06 Packages

Here is a list of packages that may be useful along with the default install:

#7 34.06 - `python#.#-venv`: provides the standard library `venv` module

#7 34.06 - `python#.#-lib2to3`: provides the `2to3-#.#` utility as well as the standard library `lib2to3` module
#7 34.06 - `python#.#-gdbm`: provides the standard library `dbm.gnu` module
#7 34.06 - `python#.#-tk`: provides the standard library `tkinter` module
