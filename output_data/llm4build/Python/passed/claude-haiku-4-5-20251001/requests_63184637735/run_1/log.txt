#7 102.6 Packages
#7 102.6 ========
#7 102.6 
#7 102.6 The packages provided here are loosely based on the debian upstream packages with some modifications to make them more usable as non-default pythons and on ubuntu. Here is a list of packages that may be useful along with the default install:
#7 102.6 
#7 102.6 - `python#.#-dev`: includes development headers for building C extensions
#7 102.6 - `python#.#-venv`: provides the standard library `venv` module
#7 102.6 - `python#.#-distutils`: provides the standard library `distutils` module
#7 102.6 - `python#.#-lib2to3`: provides the `2to3-#.#` utility as well as the standard library `lib2to3` module
#7 102.6 - `python#.#-gdbm`: provides the standard library `dbm.gnu` module
#7 102.6 - `python#.#-tk`: provides the standard library `tkinter` module
