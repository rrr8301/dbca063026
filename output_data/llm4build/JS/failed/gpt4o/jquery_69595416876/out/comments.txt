  [Root cause classification]
  - The LLM used the deprecated chromedriver.storage.googleapis.com LATEST_RELEASE API, which was shut down by Google in 2023 in favour of the Chrome for Testing (CfT) JSON endpoints. The curl call returns an XML 404 error body instead of a version string, causing wget to attempt to download that XML text as a URL — resulting in cascading host-resolution failures. The correct approach is to use https://googlechromelabs.github.io/chrome-for-testing/known-good-versions-with-downloads.json to match the installed Chrome version to its ChromeDriver binary.
